# Deep Learning by Doing Camp

这是我的深度学习学习项目。

本项目采用 **learning by doing** 的方式，不只是阅读资料，而是把学习过程持续整理成：

- 系统化学习日志
- 数学基础笔记
- Python / PyTorch 代码实践
- 深度学习模型笔记
- 论文阅读记录
- 可复现实验
- 未来可扩展成课程或集训营的材料

## 当前阶段：Week 0

Week 0 的核心任务是搭建基础设施。

- GitHub 仓库
- 服务器端开发环境
- MkDocs Material 文档站
- GitHub Pages 发布流程
- 第一篇线性代数复习记录
- 第一个 Python 数值计算脚本

## 项目结构

```text
deep-learning-by-doing-camp/
├── configs/        # 配置文件
├── docs/           # 学习文档与网页内容
├── notebooks/      # 探索性 notebook
├── reports/        # 阶段性报告
├── results/        # 运行结果
├── scripts/        # 可执行脚本
├── src/            # 可复用源码
├── tests/          # 测试代码
├── requirements.txt
├── pyproject.toml
└── mkdocs.yml
```

## 学习原则

1. 每周至少完成一次学习记录。
2. 每个核心知识点尽量配套一个最小代码实验。
3. 数学笔记、代码、图示、总结统一进入本仓库。
4. 所有阶段性成果都通过 Git commit 记录。
5. 优先追求可持续积累，而不是一次性做完。
